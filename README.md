## Architektura Systemu (Core Capabilities)

Zdekonstruowaliśmy tradycyjny model zarządzania hotelem i zbudowaliśmy go od zera, opierając się na czterech absolutnych filarach:

### 01. Silnik Rezerwacyjny (Smart Booking)
Zarządzanie przepływem gości bez najmniejszego oporu. Algorytmy G6 dbają o perfekcyjną alokację przestrzeni.
* **Skalowalna Kategoryzacja:** Płynna obsługa rezerwacji wielowęzłowych (Grupy, Rodziny) oraz mikrozarządzanie (Pary, Single).
* **Spatial Availability Matrix:** Błyskawiczne, wielowątkowe sprawdzanie dostępności terminów i pokoi w czasie rzeczywistym. Zero podwójnych rezerwacji.
* **Zero-Friction Rejections:** Zautomatyzowane, eleganckie protokoły odmowy rezerwacji w przypadku braku zasobów.

### 02. Dynamiczna Matryca Cenowa (Yield Management)
System nie tylko przechowuje ceny – on nimi oddycha.
* **Architektura Zniżek:** Globalne i lokalne zarządzanie regułami promocyjnymi. 
* **Liquid Control:** Błyskawiczne dodawanie, modyfikacja i wygaszanie kampanii zniżkowych bez zatrzymywania pracy systemu.

### 03. Ekosystem Doświadczeń (Amenities & Dining)
Pobyt to nie tylko pokój. To całe spektrum odczuć, nad którym system przejmuje pełną kontrolę.
* **Moduł Gastronomiczny (Dining OS):** Zarządzanie formami wyżywienia, dedykowanymi zniżkami restauracyjnymi i cyrkulacją nowości w menu.
* **Strefa Wellness:** Zintegrowana kontrola dostępu i harmonogramowania przestrzeni luksusowych (SPA, Basen).

### 04. Terminal Transakcyjny
* **Seamless Payments:** Architektura płatności wbudowana w rdzeń systemu. Czysto, bezpiecznie, bez przekierowań, które niszczą zaufanie.

## 🛠 Instalacja (Zero Config)

Uruchomienie środowiska G6 to kwestia sekund, nie godzin. Szanujemy Twój czas.

```bash
# 1. Sklonuj repozytorium
git clone [https://github.com/TwojProfil/G6_Hotel.git](https://github.com/TwojProfil/G6_Hotel.git)

# 2. Wejdź do matrycy
cd G6_Hotel

# 3. Zainstaluj zależności
npm install

# 4. Uruchom środowisko deweloperskie
npm run dev
